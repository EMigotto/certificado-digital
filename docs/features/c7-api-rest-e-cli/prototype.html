Usage Examples

cURL

curl -H "Authorization: Bearer st_xxxxx..." \
  https://api.example.com/api/certificates

CLI

export CERTIFICADO_TOKEN=st_xxxxx...
export CERTIFICADO_API_URL=https://api.example.com

certificado-cli certs list
certificado-cli certs get <id> --export pem > cert.pem
certificado-cli certs renew <id>

Python

import requests

headers = {
    "Authorization": f"Bearer st_xxxxx..."
}
response = requests.get(
    "https://api.example.com/api/certificates",
    headers=headers
)
print(response.json())

API Documentation

Full API documentation is available at /api/docs with Swagger UI.

Endpoints Overview

Method Endpoint Description Auth Scope
GET /api/certificates List all certificates (paginated) cert:read
GET /api/certificates/:id Get certificate details cert:read
POST /api/certificates Create new certificate cert:create
PATCH /api/certificates/:id Update certificate metadata cert:update
DELETE /api/certificates/:id Soft-delete (revoke) certificate cert:delete
POST /api/csr Generate CSR and private key cert:csr
POST /api/certificates/:id/renew Renew a certificate cert:renew
POST /api/certificates/:id/revoke Revoke a certificate cert:revoke
GET /api/policies List policies policy:read
GET /api/zones List zones zone:read
POST /api/tokens Create service token User session only

CLI Installation & Usage

Installation

macOS / Linux

tar -xzf certificado-cli-1.0.0-linux-x64.tar.gz
sudo mv certificado-cli /usr/local/bin/
certificado-cli --version

Windows

Expand-Archive certificado-cli-1.0.0-windows-x64.zip -DestinationPath C:\Program Files\
certificado-cli --version

Configuration

certificado-cli config set api_url https://api.example.com
certificado-cli config set token st_xxxxx...

Common Commands

List Certificates

certificado-cli certs list
certificado-cli certs list --filter-status EXPIRING_SOON
certificado-cli certs list --format json | jq .

Get Certificate

certificado-cli certs get abc123
certificado-cli certs get abc123 --export pem > cert.pem
certificado-cli certs get abc123 --format json | jq .

Generate CSR

certificado-cli csr generate --cn api.example.com --key-size 2048
certificado-cli csr generate \
  --cn api.example.com \
  --org "My Company" \
  --country US \
  --state CA \
  --output-csr csr.pem \
  --output-key key.pem

Renew Certificate

certificado-cli certs renew abc123 --dry-run
certificado-cli certs renew abc123

Revoke Certificate

certificado-cli certs revoke abc123 --reason compromised
certificado-cli certs revoke abc123 --reason superseded --comment "Replaced by XYZ"
